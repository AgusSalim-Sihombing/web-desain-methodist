CSS EKSTERNAL

CSS Eksternal adalah cara untuk menambahkan gaya pada halaman web dengan menggunakan file CSS terpisah. Hal ini memungkinkan pemisahan antara konten HTML dan gaya, sehingga mempermudah pengelolaan kode.

        Contoh Penggunaan :
        <head>
            <meta charset="UTF-8">
            <link rel="stylesheet" href="style.css">
        <head>
