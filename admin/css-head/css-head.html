CSS Head

This is a sample HTML document to demonstrate CSS in the head section.

        Use of CSS in the <head> section of an HTML document allows you to define styles that will be applied to the entire document. 
        This is useful for maintaining a consistent look and feel across your web pages.
        Ex:
        <head>
            <style>
                body {
                    font-family: Arial, sans-serif;
                    background-color: #f0f0f0;
                    color: #333;
                    margin: 20px;
                }

                h1 {
                    color: #007BFF;
                    text-align: center;
                }

                p {
                    font-size: 16px;
                }
            </style>
        <head>
