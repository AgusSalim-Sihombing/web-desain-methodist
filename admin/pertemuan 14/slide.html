CSS Positioning & Backgrounds

Kita bakal bahas positioning value, properti position (top/right/bottom/left/clip), implementasi real, dan seluruh background properties yang sering dipakai.

  • Position: static, relative, fixed, absolute, sticky
  • Offsets: top/right/bottom/left + clip
  • Backgrounds: color, image, size, clip, origin

4) Positioning Value — Overview

  • static: default, elemen ikut alur dokumen, offset diabaikan
  • relative: tetap di flow, tapi bisa digeser pakai top/right/bottom/left
  • absolute: keluar dari flow; diposisikan relatif ke nearest positioned ancestor
  • fixed: nempel ke viewport (tetap saat scroll)
  • sticky: hybrid relative + fixed (menempel setelah melewati threshold)
.static { position: static; }
.relative { position: relative; top: 8px; left: 8px; }
.absolute { position: absolute; top: 12px; right: 12px; }
.fixed { position: fixed; bottom: 16px; right: 16px; }
.sticky { position: sticky; top: 0; }

5) Position Property — Offsets & Clip

Offsets

  • top, right, bottom, left bekerja kalau position ≠ static
  • Absolute: referensi ke ancestor yang diposisikan (relative/ absolute/fixed)
  • Fixed: referensi ke viewport

Clip

clip (deprecated) diganti clip-path. Tapi untuk materi klasik: clip: rect(t,r,b,l) di elemen position: absolute.

.avatar-badge{

 position:absolute; top:-6px; right:-6px;
}
.fixed-help{
 position:fixed; bottom:20px; right:20px;
}
/* Legacy */
.crop { position:absolute; clip: rect(0, 120px, 120px, 0); }

7) Background Property

  • background-color
  • background-image (+ multiple) 
    background-image: url(hero.jpg), lineargradient(#0008,#0000);
  • background-repeat, background-position, background-size
  • background-attachment (scroll/fixed/local)
  • background-clip, background-origin
  • Shorthand: background: color image position / size repeat origin clip attachment;
    • 4 
.hero{
 background: #0b1220 url('img/pattern.svg') center/cover no-repeat fixed;
}

8) CSS Background Color

Atur warna latar elemen, bisa dipadukan dengan gradient.

.pill{ background-color:#111827; }
.btn{ background-image: linear-gradient(90deg, #22d3ee, #a78bfa); }

9) Background Image

.banner{
 background-image:url('img/hero.jpg');
 background-repeat:no-repeat; /* or repeat-x */
 background-position:center 20%;
}
/* Multiple layers */
.banner.overlay{
 background-image: linear-gradient(#0008,#0002), url('img/hero.jpg');
}

10) Background Size

5
  • auto: ukuran asli
  • cover: isi penuh, potong sisi yang berlebih
  • contain: seluruh gambar terlihat, bisa ada letterbox
  • Shorthand: background: ... center / cover ... 
.cover{ background-size: cover; }
.contain{ background-size: contain; }

11) Background Clip & Origin

background-clip: area pemotongan background.

  • border-box (default)
  • padding-box
  • content-box
  • Bonus: text (WebKit) → efek teks bertekstur

background-origin: titik referensi border-box | padding-box | content-box 

.chip{ background-clip: padding-box; }
.hero-text{
 -webkit-background-clip: text;
 color: transparent;
 background-image: linear-gradient(90deg,#22d3ee,#a78bfa);
}

Recap + Tips

  • Gunakan relative di parent saat butuh child absolute
  • Sticky butuh threshold top
  • Optimalkan gambar (size & format) untuk background
  • Untuk cropping modern pakai clip-path
